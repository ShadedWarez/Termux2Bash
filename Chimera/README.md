# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Chimera/install | bash && chimera
```
## Login
```
chimera
```
## Logout
```
exit
```
## Uninstall
```
uninstall-chimera
```
## Reinstall
```
pd reinstall chimera
```
## Rename
```
pd rename chimera (new name)
```
## Restore
```
pd restore chimera
```
## Reset 
```
pd reset chimera
```
## Backup 
```
pd backup chimera
```
