# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Deepin/install | bash && deepin
```
## Login
```
deepin
```
## Logout
```
exit
```
## Uninstall
```
uninstall-deepin
```
## Reinstall
```
pd reinstall deepin
```
## Rename
```
pd rename deepin (new name)
```
## Restore
```
pd restore deepin
```
## Reset 
```
pd reset deepin
```
## Backup 
```
pd backup deepin
```
