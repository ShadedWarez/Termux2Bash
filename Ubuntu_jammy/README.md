# Install
```diff
-; apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Ubuntu_jammy/install | bash && ujammy
```
## Login
```
ujammy
```
## Logout
```
exit
```
## Uninstall
```
uninstall-ujammy
```
## Reinstall
```
pd reinstall ujammy
```
## Rename
```
pd rename ujammy (new name)
```
## Restore
```
pd restore ujammy
```
## Reset 
```
pd reset ujammy
```
## Backup 
```
pd backup ujammy
```
