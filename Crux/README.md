# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Crux/install | bash && crux
```
## Login
```
crux
```
## Logout
```
exit
```
## Uninstall
```
uninstall-crux
```
## Reinstall
```
pd reinstall crux
```
## Rename
```
pd rename crux (new name)
```
## Restore
```
pd restore crux
```
## Reset 
```
pd reset crux
```
## Backup 
```
pd backup crux
```
