# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Openwrt/install | bash && openwrt
```
## Login
```
openwrt
```
## Logout
```
exit
```
## Uninstall
```
uninstall-openwrt
```
## Reinstall
```
pd reinstall openwrt
```
## Rename
```
pd rename openwrt (new name)
```
## Restore
```
pd restore openwrt
```
## Reset 
```
pd reset openwrt
```
## Backup 
```
pd backup openwrt
```
