# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Artix/install | bash && artix
```
## Login
```
artix
```
## Logout
```
exit
```
## Uninstall
```
uninstall-artix
```
## Reinstall
```
pd reinstall artix
```
## Rename
```
pd rename artix (new name)
```
## Restore
```
pd restore artix
```
## Reset 
```
pd reset artix
```
## Backup 
```
pd backup artix
```
