# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Pardus/install | bash && pardus
```
## Login
```
pardus
```
## Logout
```
exit
```
## Uninstall
```
uninstall-pardus
```
## Reinstall
```
pd reinstall pardus
```
## Rename
```
pd rename pardus (new name)
```
## Restore
```
pd restore pardus
```
## Reset 
```
pd reset pardus
```
## Backup 
```
pd backup pardus
```
