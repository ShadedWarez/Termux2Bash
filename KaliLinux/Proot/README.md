# Install
```diff
-; apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/KaliLinu/Proot/minimal | bash && kali
```
# Login
```
kali 
```
# Logout
```
exit
```
# Uninstall
```
uninstall-kali
```
# Reinstall
```
pd reinstall kali
```
# Rename
```
pd rename kali (new name)
```
# Restore
```
pd restore kali
```
# Reset 
```
pd reset kali 
```
# Backup 
```
pd backup kali
```
