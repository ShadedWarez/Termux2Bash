# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Manjaro/install | bash && manjaro
```
## Login
```
manjaro
```
## Logout
```
exit
```
## Uninstall
```
uninstall-manjaro
```
## Reinstall
```
pd reinstall manjaro
```
## Rename
```
pd rename manjaro (new name)
```
## Restore
```
pd restore manjaro
```
## Reset 
```
pd reset manjaro
```
## Backup 
```
pd backup manjaro
```
