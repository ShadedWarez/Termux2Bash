# Install
```diff
-; apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Rocky/install | bash && rocky
```
## Login
```
rocky 
```
## Logout
```
exit
```
## Uninstall
```
uninstall-rocky
```
## Reinstall
```
pd reinstall rocky
```
## Rename
```
pd rename rocky (new name)
```
## Restore
```
pd restore rocky
```
## Reset 
```
pd reset rocky
```
## Backup 
```
pd backup rocky
```
