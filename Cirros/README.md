# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Cirros/install | bash && cirros
```
# Login
```
cirros
```
# Logout
```
exit
```
# Uninstall
```
uninstall-cirros
```
# Reinstall
```
pd reinstall cirros
```
# Rename
```
pd rename cirros (new name)
```
# Restore
```
pd restore cirros
```
# Reset 
```
pd reset cirros
```
# Backup 
```
pd backup cirros
```
