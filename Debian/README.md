# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Debian/install | bash && debian
```
## Login
```
debian
```
## Logout
```
exit
```
## Uninstall
```
uninstall-debian
```
## Reinstall
```
pd reinstall debian
```
## Rename
```
pd rename debian (new name)
```
## Restore
```
pd restore debian
```
## Reset 
```
pd reset debian
```
## Backup 
```
pd backup debian
```
