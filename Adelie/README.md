# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Adelie/install | bash && adelie
```
# Login
```
adelie
```
# Logout
```
exit
```
# Uninstall
```
uninstall-adelie
```
# Reinstall
```
pd reinstall adelie
```
# Rename
```
pd rename adelie (new name)
```
# Restore
```
pd restore adelie
```
# Reset 
```
pd reset adelie
```
# Backup 
```
pd backup adelie
```
