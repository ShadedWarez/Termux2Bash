# Install
```diff
-; apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Ufocal/install | bash && ufocal
```
## Login
```
ufocal 
```
## Logout
```
exit
```
## Uninstall
```
uninstall-ufocal
```
## Reinstall
```
pd reinstall ufocal
```
## Rename
```
pd rename ufocal (new name)
```
## Restore
```
pd restore ufocal
```
## Reset 
```
pd reset ufocal
```
## Backup 
```
pd backup ufocal
```
