# Install
```diff
-; apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/ParrotSec/install | bash && parrot
```
## Login
```
parrot 
```
## Logout
```
exit
```
## Uninstall
```
uninstall-parrot
```
## Reinstall
```
pd reinstall parrot
```
## Rename
```
pd rename parrot (new name)
```
## Restore
```
pd restore parrot 
```
## Reset 
```
pd reset parrot
```
## Backup 
```
pd backup parrot
```
