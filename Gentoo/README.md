# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Gentoo/install | bash && gentoo
```
## Login
```
gentoo
```
## Logout
```
exit
```
## Uninstall
```
uninstall-gentoo
```
## Reinstall
```
pd reinstall gentoo
```
## Rename
```
pd rename gentoo (new name)
```
## Restore
```
pd restore gentoo
```
## Reset 
```
pd reset gentoo
```
## Backup 
```
pd backup gentoo
```
