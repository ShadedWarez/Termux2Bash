# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Alma/install | bash && alma
```
# Login
```
alma
```
# Logout
```
exit
```
# Uninstall
```
uninstall-alma
```
# Reinstall
```
pd reinstall alma
```
# Rename
```
pd rename alma (new name)
```
# Restore
```
pd restore alma
```
# Reset 
```
pd reset alma
```
# Backup 
```
pd backup alma
```
