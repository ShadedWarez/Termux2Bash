# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/BackBox/install | bash && backbox
```
## Login
```
backbox
```
## Logout
```
exit
```
## Uninstall
```
uninstall-backbox
```
## Reinstall
```
pd reinstall backbox
```
## Rename
```
pd rename backbox (new name)
```
## Restore
```
pd restore backbox
```
## Reset 
```
pd reset backbox
```
## Backup 
```
pd backup backbox
```
