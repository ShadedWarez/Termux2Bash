# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Oracle/install | bash && oracle
```
## Login
```
oracle 
```
## Logout
```
exit
```
## Uninstall
```
uninstall-oracle
```
## Reinstall
```
pd reinstall oracle 
```
## Rename
```
pd rename oracle (new name)
```
## Restore
```
pd restore oracle 
```
## Reset 
```
pd reset oracle 
```
## Backup 
```
pd backup oracle 
```
