# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/centos/install | bash && centos
```
## Login
```
centos
```
## Logout
```
exit
```
## Uninstall
```
uninstall-centos
```
## Reinstall
```
pd reinstall centos
```
## Rename
```
pd rename centos (new name)
```
## Restore
```
pd restore centos
```
## Reset 
```
pd reset centos
```
## Backup 
```
pd backup centos
```
