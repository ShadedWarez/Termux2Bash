# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Fedora/install | bash && fedora 
```
## Login
```
fedora 
```
## Logout
```
exit
```
## Uninstall
```
uninstall-fedora
```
## Reinstall
```
pd reinstall fedora
```
## Rename
```
pd rename fedora (new name)
```
## Restore
```
pd restore fedora
```
## Reset 
```
pd reset fedora
```
## Backup 
```
pd backup fedora
```
