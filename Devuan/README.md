# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Devuan/install | bash && devuan
```
## Login
```
devuan
```
## Logout
```
exit
```
## Uninstall
```
uninstall-devuan
```
## Reinstall
```
pd reinstall devuan
```
## Rename
```
pd rename devuan (new name)
```
## Restore
```
pd restore devuan
```
## Reset 
```
pd reset devuan
```
## Backup 
```
pd backup devuan
```
