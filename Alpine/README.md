# Install
```
apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Alpine/install | bash && alpine
```
## Login
```
alpine 
```
## Logout
```
exit
```
## Uninstall
```
uninstall-alpine
```
## Reinstall
```
pd reinstall alpine 
```
## Rename
```
pd rename alpine (new name)
```
## Restore
```
pd restore alpine
```
## Reset 
```
pd reset alpine
```
## Backup 
```
pd backup alpine
```
