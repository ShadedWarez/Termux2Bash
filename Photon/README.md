# Install
```diff
-; apt update && apt install wget -y && clear && wget -qO- https://raw.githubusercontent.com/xiv3r/Termux-Pentesting-Distro/refs/heads/main/Photon/install | bash && photon
```
## Login
```
photon
```
## Logout
```
exit
```
## Uninstall
```
uninstall-photon
```
## Reinstall
```
pd reinstall photon
```
## Rename
```
pd rename photon (new name)
```
## Restore
```
pd restore photon
```
## Reset 
```
pd reset photon
```
## Backup 
```
pd backup photon
```
